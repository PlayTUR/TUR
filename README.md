# The Unnamed Rhythm Game (TUR)

A Python-based rhythm game with custom song support, editor, and multiplayer.

## Installation

1. **Install Python 3.8+**
2. **Install Dependencies:**
   ```bash
   pip install -r requirements.txt
   ```
   (This includes `pygame`, `numpy`, `ngrok`, etc.)

## How to Play

Run the game:
```bash
python main.py
```

### Controls
- **Arrows / WASD**: Select in menus
- **D F J K**: Hit notes (customizable)
- **ENTER**: Confirm
- **ESC**: Back / Pause

## Multiplayer Guide

TUR supports two multiplayer modes:

### 1. LAN Mode (Local Network)
Best for playing with friends on the same WiFi. **Zero setup required.**

*   **Host:** Select **Multiplayer -> HOST (LAN)**.
*   **Join:** Select **Multiplayer -> JOIN (LAN)**. The game will automatically find the host.

### 2. NGROK Mode (Online)
Play with anyone over the internet without port forwarding.

**One-Time Setup (Host Only):**
1.  Create a free account at [https://ngrok.com](https://ngrok.com).
2.  Get your **Authtoken** from the ngrok dashboard.

**Hosting:**
1.  Select **Multiplayer -> HOST (NGROK)**.
2.  Enter your ngrok Authtoken (saved for next time).
3.  Share the **Room Code** (e.g., `812345` or `EU-123456`) with your friend.

**Joining:**
1.  Select **Multiplayer -> JOIN (NGROK)**.
2.  **No account needed.** Just enter the Room Code provided by the host.

## Adding Songs
Place your song folders in the `songs/` directory. Each folder should contain:
- `audio.mp3` or `audio.ogg`
- `chart.json` (generated by the in-game editor)
